Wearable apps should specify whether they can work standalone, without a phone app. Add a valid meta-data entry for com.google.android.wearable.standalone to your application element and set the value to true or false. <meta-data android:name="com.google.android.wearable.standalone" android:value="true"/>`

More information: https://developer.android.com/training/wearables/apps/packaging.html
